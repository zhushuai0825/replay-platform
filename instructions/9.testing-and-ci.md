# 9. 测试与 CI 基础

1. **后端测试**
   - 使用 pytest + httpx/TestClient
   - 典型测试：任务列表分页、创建任务的参数校验、文件上传限制
   - 运行：`pytest`
2. **前端测试（可选）**
   - 单元：Vitest + Vue Testing Library
   - E2E：Playwright 或 Cypress（模拟上传/任务创建流程）
3. **CI 工作流（以 GitHub Actions 为例）**
   - `.github/workflows/ci.yml`
   - 步骤：checkout → setup Python → install deps → pytest → setup Node → npm install/test → docker compose build (可选)
4. **lint/format**
   - 后端：black + isort + flake8
   - 前端：ESLint + Prettier
   - 在 CI 中添加相应步骤，确保提交前通过
5. **Artifacts & 报告**
   - pytest --junitxml 输出测试报告
   - 前端构建产物或日志可通过 Actions artifacts 保存
