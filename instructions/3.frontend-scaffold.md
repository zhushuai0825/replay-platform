# 3. 前端脚手架（Vue 3 + Vite）

1. 确认 Node 18+ 可用：`node -v`、`npm -v`
2. 进入 `replay-platform/frontend`，执行：`npm create vite@latest . -- --template vue`
3. 安装依赖：`npm install`
4. 目录结构建议：
   ```
   frontend/
     src/
       api/index.js
       views/{Dashboard,Tasks,Traffic,Environments}.vue
       router/index.js
       store/
   ```
5. 配置 axios 客户端 `src/api/index.js`
   - 创建 axios 实例 (`baseURL = import.meta.env.VITE_API_BASE_URL || 'http://localhost:8000'`)
   - 请求拦截器：读取 localStorage token 附加 Authorization 头
   - 响应拦截器：返回 `response.data`，统一处理错误
6. 创建基础页面骨架
   - 每个 `.vue` 文件包含 `<template>` + `<script setup>` + `<style>`，先展示占位文字
7. 配置路由 `src/router/index.js`
   - 使用 `createRouter` / `createWebHistory`
   - 定义 `/dashboard`、`/tasks`、`/traffic`、`/environments`
   - 在 `src/main.js` 中 `app.use(router)` 后 `app.mount('#app')`
8. 本地运行
   - `npm run dev`
   - 浏览器访问 `http://localhost:5173`
9. 生产构建
   - `npm run build` -> 产物在 `dist/`
   - 预留 `Dockerfile`：使用 `node:18-alpine` 构建 + `nginx:alpine` 托管
