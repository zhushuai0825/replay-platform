# 17. 前端状态管理与 UI 规范

1. 状态管理
   - 使用 Pinia 创建 `useTaskStore`、`useEnvironmentStore`
   - 统一管理任务列表、分页信息、当前任务详情
2. UI 规范
   - 选定 UI 库（Element Plus / Ant Design Vue），统一色彩与组件
   - 自定义布局：侧边导航 + 顶部 Header + 内容区域
3. 组件划分
   - 通用组件：搜索栏、分页器、信息卡、进度条
   - 业务组件：任务列表、任务详情、流量上传面板
4. 表单与校验
   - 使用 Element Plus Form + yup 或自行校验
   - 对必填项、URL、文件类型做前端校验
5. 国际化（可选）
   - 引入 vue-i18n，为标题/提示语提供多语言支持
