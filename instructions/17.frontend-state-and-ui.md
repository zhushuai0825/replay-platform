# 17. 前端状态管理与 UI 规范（当前状态）

---

## 1. 已完成的内容

- **Pinia 集成**
  - 依赖：`pinia` 已写入 `package.json`
  - `src/main.js` 中创建并挂载 Pinia
- **任务 Store**
  - `src/store/taskStore.js` 定义 `useTaskStore`，负责：
    - 保存 `items / total / page / pageSize / loading / error`
    - 调用 `GET /api/v1/tasks/` 获取数据
    - 提供 `setPage()`、`fetchTasks()` 方法
  - `src/store/index.js` 统一导出 store，兼容 `@/store`
- **任务页面**
  - `src/views/Tasks.vue` 使用 Pinia 数据，显示表格 + 分页 + 刷新按钮
  - 支持加载状态、错误提示、分页切换
