# 1. 环境准备

1. 安装 Homebrew（若已安装可跳过）
   - 打开终端运行：`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
2. 通过 Homebrew 安装基础工具
   - `brew install python@3.11 node@18 git docker docker-compose`
   - 如已安装，请确保版本满足：Python>=3.11，Node>=18，Docker Desktop>=4.28
3. 启动 Docker Desktop 并等待引擎就绪
   - 打开应用：`open -a Docker`
   - 运行 `docker info` 无错误即表示可用
4. 规划项目目录结构
   - 在 `replay-platform/` 下准备以下空目录（后续步骤会填充）：
     ```
     replay-platform/
       backend/
       frontend/
       docs/
       scripts/
     ```
5. 准备 Git 仓库（可选）
   - 在 `replay-platform/` 目录运行 `git init`
   - 创建 `.gitignore` 文件（已自动生成，包含以下忽略规则）：
     - Python：虚拟环境、`__pycache__`、`.pyc`、构建产物等
     - Node.js：`node_modules/`、日志、构建产物等
     - IDE：`.vscode/`、`.idea/`、`.DS_Store` 等
     - 环境变量：`.env`、`.env.local` 等（避免泄露敏感信息）
     - 日志：`*.log`、`logs/` 目录
     - 数据库：`*.db`、`*.sqlite` 等
     - 数据文件：`backend/data/traffic/*`、`backend/data/temp/*`（但保留 `.gitkeep`）
     - 测试：`.pytest_cache/`、`.coverage` 等
   - 如果 `.gitignore` 文件不存在，可以手动创建并复制上述规则
