# 13. 结果比对与任务详情实现

1. 数据模型
   - `ReplayResult`：`task_id`、`request_index`、`original_request/response`、`replayed_request/response`、`status`、`diff_details`、`response_time`
2. 比对流程（示例）
   - 请求/响应解析 → 忽略字段（配置）→ JSON diff → 记录差异
3. API
   - `GET /api/v1/tasks/{id}`：返回任务基础信息（环境、流量文件、统计）
   - `GET /api/v1/tasks/{id}/results`：分页 + status 过滤 + diff_details
4. 前端展示
   - 任务详情页：信息卡、进度条、统计、日志
   - 结果列表：状态筛选（成功/失败/diff），支持查看 diff 详情
5. 扩展
   - diff 规则配置（JSONPath/正则）
   - diff 导出、任务报告
