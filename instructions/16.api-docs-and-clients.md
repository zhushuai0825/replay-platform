## 1. 自动文档

- FastAPI 自带：
  - Swagger UI：`/docs`
  - ReDoc：`/redoc`
- 路由中已经使用 `tags`、`summary`、`response_model`
- `/metrics` 提供 Prometheus 指标

---

## 2. 手工文档

- 新增 `docs/api.md`，记录现有接口（任务、流量、系统查询）
- 每次新增接口后，请同步更新该文件

---

## 3. Postman / Thunder Client

- 建议创建环境变量：
  ```
  base_url = http://localhost:8000
  ```
- 常用请求：
  - GET `{{base_url}}/api/v1/tasks/?page=1&page_size=10`
  - POST `{{base_url}}/api/v1/tasks/`
  - POST `{{base_url}}/api/v1/traffic/upload`

---
