# 16. API 文档与调试工具

1. 自动文档
   - FastAPI 自带 Swagger `/docs`、ReDoc `/redoc`
   - 通过 `tags`、`summary`、`description`、`response_model` 提升可读性
2. 手工文档
   - 在 `docs/api.md` 中记录关键接口、参数、示例
   - 标注鉴权方式、返回字段说明
3. Postman / Thunder Client
   - 创建环境变量（base_url、token）
   - 保存常用请求：任务列表、创建任务、上传流量、环境创建
4. Mock & Sandbox
   - 在早期可用 Postman mock server/JSON Server 模拟后端
   - 前端可先对接 mock，再切换到真实接口
5. SDK/客户端
   - 如有需要，可以封装 Python/JS 客户端，便于其他服务调用
